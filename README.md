# 个人做的一个图片分享网站
## 数据是从网上爬取然后保存在本地的，没有用到后端http请求。

>4种图片分类，动漫、游戏、风景、动物。

## 一些辅助功能：分页、按图片简介搜索、幻灯片、放大
